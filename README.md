# my-vue-app

这是一个基于 Vue 3 + Vite 的前端项目，当前主要包含登录、注册、首页、仪表盘等页面，以及用户状态管理和接口请求封装。

## 技术栈

- Vue 3
- Vite
- TypeScript
- Vue Router
- Pinia
- Axios
- Element Plus

## 项目结构

- `src/main.ts`：应用入口
- `src/router/`：路由配置
- `src/stores/`：Pinia 状态管理
- `src/utils/axios.ts`：接口请求封装
- `src/views/`：页面组件
- `src/components/`：通用组件
- `src/assets/styles/`：全局样式

## 本地运行

```bash
npm install
npm run dev
```

## 构建与预览

```bash
npm run build
npm run preview
```
